# Koncepthive Gym Website

## Tech Stack
- **Frontend:** React, Next.js
- **Styling:** TailwindCSS, CSS Variables  
- **Animations & Interactivity:** Framer Motion, React Hot Toast

## Setup Instructions

1. **Clone the repository**
```bash

cd gym-website

2. **Install dependencies**
```bash
npm install

3. **Start Development Server**
```bash
npm run dev

4. **Production Build**
```bash
